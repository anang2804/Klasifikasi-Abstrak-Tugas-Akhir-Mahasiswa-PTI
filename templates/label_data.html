Tentang Data Latih
Apa itu Data Latih?

Data latih adalah data abstrak yang sudah dilabel (RPL atau TKJ) dan digunakan untuk melatih model KNN. Data ini diperoleh dari scraping ejournal.unesa.ac.id dan dilabel secara otomatis menggunakan keyword scoring.

Perbedaan dengan Data Uji:
Data Latih (Training Data)
  • Dari scraping ejournal.unesa.ac.id
  • Label otomatis dengan keyword scoring
  • Digunakan untuk melatih model
  • Total: {{ stats.total }} data
Data Uji (Test Data)
  • Dari input manual & upload file
  • Label prediksi dari model KNN
  • Digunakan untuk evaluasi & tracking
  • Lihat di Menu Data Uji
